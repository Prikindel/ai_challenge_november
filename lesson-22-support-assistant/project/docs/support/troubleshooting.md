# Troubleshooting - Решение проблем

## Проблемы с авторизацией

### Ошибка "Неверный email или пароль"

**Причины:**
- Неправильно введён email или пароль
- Аккаунт заблокирован
- Пароль был изменён

**Решение:**
1. Проверьте правильность ввода email и пароля
2. Попробуйте восстановить пароль
3. Проверьте статус аккаунта в настройках
4. Очистите кэш браузера

### Ошибка "Аккаунт заблокирован"

**Причины:**
- Множественные неудачные попытки входа
- Нарушение правил использования сервиса
- Подозрительная активность

**Решение:**
1. Свяжитесь с поддержкой для разблокировки
2. Предоставьте информацию для идентификации
3. Дождитесь проверки и разблокировки аккаунта

### Ошибка "Сессия истекла"

**Причины:**
- Долгое бездействие
- Вход с другого устройства
- Изменение пароля

**Решение:**
1. Войдите в аккаунт заново
2. Включите "Запомнить меня" для длительных сессий
3. Проверьте настройки безопасности

## Проблемы с подпиской

### Подписка не активируется

**Причины:**
- Проблемы с платёжной системой
- Недостаточно средств на счёте
- Ошибка обработки платежа

**Решение:**
1. Проверьте статус платежа в истории платежей
2. Убедитесь, что на счёте достаточно средств
3. Попробуйте другой способ оплаты
4. Свяжитесь с поддержкой, если проблема сохраняется

### Подписка не продлевается автоматически

**Причины:**
- Истёк срок действия карты
- Недостаточно средств
- Отключена автопродление

**Решение:**
1. Проверьте данные карты в настройках
2. Убедитесь, что включено автопродление
3. Пополните счёт
4. Обновите данные карты при необходимости

## Проблемы с производительностью

### Медленная загрузка страниц

**Причины:**
- Медленное интернет-соединение
- Перегрузка сервера
- Проблемы с браузером

**Решение:**
1. Проверьте скорость интернета
2. Очистите кэш браузера
3. Закройте лишние вкладки
4. Попробуйте другой браузер
5. Отключите расширения браузера

### Ошибка "Превышено время ожидания"

**Причины:**
- Медленное соединение
- Перегрузка сервера
- Проблемы с сетью

**Решение:**
1. Проверьте интернет-соединение
2. Обновите страницу
3. Попробуйте позже
4. Используйте проводное соединение вместо Wi-Fi

## Проблемы с данными

### Данные не сохраняются

**Причины:**
- Проблемы с браузером
- Отключён JavaScript
- Проблемы с cookies

**Решение:**
1. Включите JavaScript в браузере
2. Разрешите использование cookies
3. Очистите кэш и cookies
4. Попробуйте другой браузер

### Данные не синхронизируются

**Причины:**
- Проблемы с интернет-соединением
- Конфликт версий
- Проблемы с сервером

**Решение:**
1. Проверьте интернет-соединение
2. Обновите страницу
3. Выйдите и войдите в аккаунт заново
4. Свяжитесь с поддержкой

## Проблемы с файлами

### Файлы не загружаются

**Причины:**
- Превышен размер файла
- Неподдерживаемый формат
- Проблемы с соединением

**Решение:**
1. Проверьте размер файла (максимум 100 МБ)
2. Проверьте формат файла
3. Сожмите файл, если он слишком большой
4. Попробуйте загрузить файл позже

### Файлы не скачиваются

**Причины:**
- Блокировка браузером
- Проблемы с правами доступа
- Повреждённый файл

**Решение:**
1. Проверьте настройки браузера
2. Разрешите скачивание файлов
3. Попробуйте другой браузер
4. Свяжитесь с поддержкой

## Проблемы с безопасностью

### Подозрительная активность в аккаунте

**Причины:**
- Взлом аккаунта
- Утечка пароля
- Небезопасное соединение

**Решение:**
1. Немедленно измените пароль
2. Включите двухфакторную аутентификацию
3. Проверьте активность аккаунта
4. Свяжитесь с поддержкой
5. Отключите незнакомые устройства

### Ошибка "Небезопасное соединение"

**Причины:**
- Проблемы с SSL сертификатом
- Устаревший браузер
- Проблемы с сетью

**Решение:**
1. Обновите браузер
2. Проверьте дату и время на устройстве
3. Очистите кэш SSL
4. Попробуйте другой браузер

## Общие проблемы

### Страница не отображается правильно

**Причины:**
- Устаревший браузер
- Отключён JavaScript
- Проблемы с CSS

**Решение:**
1. Обновите браузер до последней версии
2. Включите JavaScript
3. Очистите кэш
4. Отключите расширения браузера

### Функция не работает

**Причины:**
- Временная недоступность
- Проблемы с браузером
- Ошибка в коде

**Решение:**
1. Обновите страницу
2. Попробуйте позже
3. Очистите кэш
4. Сообщите о проблеме в поддержку

## Когда обращаться в поддержку

Обратитесь в поддержку, если:
- Проблема не решается описанными способами
- Вы подозреваете взлом аккаунта
- Произошла ошибка при оплате
- Данные были потеряны
- Функция критически важна и не работает

При обращении в поддержку укажите:
- Описание проблемы
- Шаги для воспроизведения
- Скриншоты ошибок
- Информацию о браузере и ОС
- Время возникновения проблемы
