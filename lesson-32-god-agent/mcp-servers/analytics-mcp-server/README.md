# Analytics MCP Server

MCP сервер для анализа данных из различных источников: CSV, JSON и SQLite баз данных.

## Инструменты

### 1. `analyze_csv`
Анализирует CSV файлы и отвечает на вопросы о данных.

**Параметры:**
- `file_path` (string) - путь к CSV файлу
- `query` (string) - вопрос для анализа

**Примеры запросов:**
- "Какие колонки есть в файле?"
- "Сколько строк в файле?"
- "Какие средние значения в числовых колонках?"

### 2. `analyze_json`
Анализирует JSON файлы и возвращает структуру данных.

**Параметры:**
- `file_path` (string) - путь к JSON файлу
- `query` (string) - вопрос для анализа

**Примеры запросов:**
- "Какая структура данных?"
- "Сколько элементов в массиве?"
- "Какие ключи в объекте?"

### 3. `analyze_database`
Анализирует SQLite базы данных. Может выполнять SQL запросы или отвечать на вопросы.

**Параметры:**
- `db_path` (string) - путь к SQLite базе данных
- `query` (string) - SQL запрос или вопрос

**Примеры запросов:**
- "Какие таблицы есть в базе?"
- "SELECT COUNT(*) FROM users"
- "Какая структура таблицы reviews?"

### 4. `get_data_summary`
Получает базовую сводку по источнику данных (CSV, JSON или БД).

**Параметры:**
- `data_source` (string) - путь к источнику данных

## Сборка

```bash
./gradlew build
```

## Запуск

```bash
./gradlew run
```

Или через JAR:
```bash
java -jar build/libs/analytics-mcp-server-1.0.0.jar
```

## Зависимости

- OpenCSV для работы с CSV файлами
- SQLite JDBC для работы с базами данных
- Kotlinx Serialization для работы с JSON
