# База знаний God Agent

Это директория содержит вашу персональную базу знаний, организованную по категориям.

## Структура

```
knowledge-base/
├── projects/      # Документация и заметки по проектам
├── learning/      # Материалы для обучения и изучения
├── personal/      # Личные заметки и цели
└── references/   # Справочная информация и шпаргалки
```

## Как использовать

### 1. Добавление документов

Просто добавьте файлы `.md` или `.txt` в соответствующую категорию:

- **projects/** - для документации проектов, архитектурных решений, заметок о разработке
- **learning/** - для конспектов, статей, учебных материалов
- **personal/** - для личных заметок, целей, планов
- **references/** - для справочной информации, шпаргалок, документации API

### 2. Индексация

После добавления документов их нужно проиндексировать:

1. Откройте страницу "База знаний" в UI
2. Нажмите "Индексировать всё" или индексируйте отдельные категории
3. Дождитесь завершения индексации

### 3. Поиск

Используйте поиск в UI или задавайте вопросы в чате:
- "Найди информацию о проекте X"
- "Что такое RAG?"
- "Покажи примеры использования API"

## Формат документов

Поддерживаются следующие форматы:
- **Markdown (.md)** - рекомендуется для структурированных документов
- **Plain text (.txt)** - для простых заметок

## Рекомендации

1. **Используйте заголовки** для структурирования документов
2. **Добавляйте метаданные** в начало документа (дата, автор, теги)
3. **Разбивайте большие документы** на логические части
4. **Используйте осмысленные имена файлов** для лучшего поиска
5. **Регулярно обновляйте** документы и переиндексируйте

## Примеры

В каждой категории есть примеры документов:
- `projects/example-project.md` - пример документации проекта
- `learning/rag-basics.md` - основы RAG
- `learning/mcp-protocol.md` - протокол MCP
- `personal/goals-2024.md` - личные цели
- `references/kotlin-cheatsheet.md` - шпаргалка по Kotlin
- `references/api-endpoints.md` - справочник API

## Автоматическая индексация

Если в конфигурации включена опция `auto_index: true`, база знаний будет автоматически индексироваться при запуске сервера.

## Мониторинг

Используйте страницу "База знаний" для:
- Просмотра статистики по категориям
- Поиска документов
- Управления индексацией
