# База знаний God Agent

Эта директория содержит вашу персональную базу знаний, которая индексируется и используется для RAG (Retrieval-Augmented Generation).

## Структура

```
knowledge-base/
├── projects/      # Документация и заметки по проектам
├── learning/       # Обучающие материалы и заметки
├── personal/       # Личные заметки, цели, идеи
└── references/     # Справочные материалы, статьи, книги
```

## Использование

1. **Добавьте документы** в соответствующие категории
2. **Индексируйте базу знаний** через UI или API (`/api/knowledge-base/index`)
3. **Используйте в чате** - God Agent автоматически найдет релевантную информацию

## Формат документов

Поддерживаются файлы:
- Markdown (`.md`)
- Текстовые файлы (`.txt`)
- Другие текстовые форматы
