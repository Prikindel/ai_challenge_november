# Example Project

Пример проекта для демонстрации базы знаний.

## Описание

Этот проект демонстрирует, как организовать документацию проекта в базе знаний God Agent.

## Архитектура

Проект использует:
- Kotlin + Ktor для backend
- React для frontend
- PostgreSQL для базы данных

## API

Основные endpoints:
- `GET /api/users` - список пользователей
- `POST /api/users` - создание пользователя
- `GET /api/users/{id}` - получение пользователя

## Деплой

Проект деплоится на VPS через Docker.
