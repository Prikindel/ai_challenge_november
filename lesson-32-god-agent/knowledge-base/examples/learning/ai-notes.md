# Заметки по AI

Заметки и мысли об искусственном интеллекте.

## RAG (Retrieval-Augmented Generation)

RAG - это техника, которая объединяет поиск информации с генерацией текста.

### Как работает RAG

1. Индексация документов в векторную базу
2. Поиск релевантных фрагментов по запросу
3. Использование найденных фрагментов как контекст для LLM
4. Генерация ответа с учетом контекста

### Преимущества

- Меньше галлюцинаций
- Актуальная информация
- Возможность цитирования источников

## MCP (Model Context Protocol)

MCP - протокол для подключения инструментов к AI агентам.

### Основные концепции

- **Tools** - инструменты, которые может использовать агент
- **Resources** - ресурсы, к которым агент может получить доступ
- **Prompts** - шаблоны промптов

### Использование

MCP позволяет создавать модульные системы, где каждый сервер предоставляет свои инструменты.
