# Примеры контента для базы знаний

Эта директория содержит примеры документов для демонстрации работы базы знаний God Agent.

## Структура

- `projects/` - примеры документации проектов
- `learning/` - примеры заметок по обучению
- `personal/` - примеры личных заметок
- `references/` - примеры справочных материалов

## Использование

Скопируйте примеры в соответствующие директории `knowledge-base/` и проиндексируйте базу знаний.

## Добавление своих документов

1. Создайте документы в формате Markdown
2. Разместите их в соответствующих категориях
3. Переиндексируйте базу знаний через UI или API
