# Личные цели на 2024 год

## Профессиональные цели

1. **Изучить AI/ML технологии**
   - Пройти курс по машинному обучению
   - Реализовать несколько проектов с использованием LLM
   - Изучить RAG и fine-tuning моделей

2. **Развить навыки разработки**
   - Изучить Kotlin на продвинутом уровне
   - Освоить архитектурные паттерны
   - Улучшить навыки работы с базами данных

3. **Создать полезные проекты**
   - Разработать God Agent до production-ready состояния
   - Создать библиотеку для работы с MCP
   - Внести вклад в open-source проекты

## Личные цели

1. **Здоровье**
   - Регулярные тренировки (3 раза в неделю)
   - Правильное питание
   - Достаточный сон (7-8 часов)

2. **Обучение**
   - Читать минимум 1 книгу в месяц
   - Изучать новые технологии
   - Посещать конференции и митапы

3. **Баланс**
   - Находить время для хобби
   - Проводить время с семьей и друзьями
   - Делать перерывы в работе

- Количество завершенных проектов: 5+
