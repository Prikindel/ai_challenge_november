# Пример проекта: God Agent

## Описание

God Agent - это универсальный AI-помощник, объединяющий множество возможностей в едином интерфейсе.

## Основные компоненты

### 1. MCP Router
Модульная система для подключения различных MCP серверов. Каждый сервер предоставляет набор инструментов, которые могут быть использованы God Agent.

### 2. RAG система
Система поиска по базе знаний с использованием эмбеддингов. Поддерживает категоризацию документов и семантический поиск.

### 3. Персонализация
Адаптация ответов под профиль пользователя, учет предпочтений и стиля общения.

### 4. Голосовой ввод
Интеграция с Vosk для распознавания речи на русском языке.

## Технологии

- Kotlin/Ktor для backend
- SQLite для хранения данных
- Ollama для локального LLM
- Vosk для распознавания речи
- MCP протокол для модульности

## Архитектура

God Agent использует модульную архитектуру:
- Каждый компонент может быть включен/выключен через конфигурацию
- MCP серверы подключаются динамически
- База знаний индексируется автоматически
