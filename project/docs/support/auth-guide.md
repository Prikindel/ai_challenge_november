# Auth Guide - Руководство по авторизации

## Вход в аккаунт

### Стандартный вход

1. Перейдите на страницу входа
2. Введите ваш email адрес
3. Введите пароль
4. (Опционально) Отметьте "Запомнить меня"
5. Нажмите "Войти"

### Вход через социальные сети

1. Перейдите на страницу входа
2. Выберите социальную сеть (Google, Facebook, GitHub)
3. Авторизуйтесь в выбранной сети
4. Предоставьте необходимые разрешения
5. Аккаунт будет создан автоматически

### Восстановление доступа

**Восстановление пароля:**
1. На странице входа нажмите "Забыли пароль?"
2. Введите ваш email
3. Проверьте почту
4. Перейдите по ссылке из письма
5. Введите новый пароль
6. Подтвердите новый пароль

**Восстановление email:**
1. Свяжитесь с поддержкой
2. Предоставьте информацию для идентификации
3. Дождитесь проверки
4. Получите инструкции по восстановлению

## Безопасность входа

### Надёжный пароль

**Требования к паролю:**
- Минимум 8 символов
- Буквы (заглавные и строчные)
- Цифры
- Специальные символы

**Рекомендации:**
- Не используйте личную информацию
- Не используйте словарные слова
- Используйте уникальный пароль для каждого сервиса
- Регулярно меняйте пароль

### Двухфакторная аутентификация (2FA)

**Настройка 2FA:**
1. Войдите в аккаунт
2. Перейдите в "Настройки" → "Безопасность"
3. Включите "Двухфакторная аутентификация"
4. Сканируйте QR-код приложением-аутентификатором
5. Введите код подтверждения
6. Сохраните резервные коды

**Использование 2FA:**
1. Введите email и пароль
2. Введите код из приложения-аутентификатора
3. Войдите в аккаунт

**Восстановление доступа при потере 2FA:**
1. Используйте резервные коды
2. Или свяжитесь с поддержкой для восстановления

### Защита сессии

**Безопасный вход:**
- Используйте HTTPS соединение
- Не входите в аккаунт на общественных компьютерах
- Выходите из аккаунта после использования
- Используйте "Запомнить меня" только на личных устройствах

**Мониторинг сессий:**
1. Перейдите в "Настройки" → "Безопасность"
2. Просмотрите список активных сессий
3. Завершите незнакомые сессии
4. Настройте уведомления о новых входах

## Проблемы с авторизацией

### Ошибка "Неверный email или пароль"

**Возможные причины:**
- Неправильно введён email или пароль
- Аккаунт заблокирован
- Пароль был изменён

**Решение:**
1. Проверьте правильность ввода
2. Попробуйте восстановить пароль
3. Проверьте статус аккаунта
4. Очистите кэш браузера

### Ошибка "Аккаунт заблокирован"

**Возможные причины:**
- Множественные неудачные попытки входа
- Нарушение правил использования
- Подозрительная активность

**Решение:**
1. Свяжитесь с поддержкой
2. Предоставьте информацию для идентификации
3. Дождитесь проверки и разблокировки

### Ошибка "Сессия истекла"

**Возможные причины:**
- Долгое бездействие
- Вход с другого устройства
- Изменение пароля

**Решение:**
1. Войдите в аккаунт заново
2. Включите "Запомнить меня" для длительных сессий
3. Проверьте настройки безопасности

## Безопасность аккаунта

### Защита от взлома

**Рекомендации:**
- Используйте надёжный пароль
- Включите двухфакторную аутентификацию
- Регулярно проверяйте активность аккаунта
- Не передавайте пароль третьим лицам
- Используйте уникальный пароль для каждого сервиса

### Что делать при взломе

1. Немедленно измените пароль
2. Включите двухфакторную аутентификацию
3. Проверьте активность аккаунта
4. Завершите все активные сессии
5. Свяжитесь с поддержкой
6. Сообщите о подозрительной активности

### Регулярная проверка безопасности

**Рекомендуется:**
- Менять пароль каждые 3-6 месяцев
- Проверять активность аккаунта еженедельно
- Обновлять резервные коды 2FA
- Проверять список активных сессий
- Обновлять информацию для восстановления

## Дополнительные настройки

### Автоматический выход

**Настройка:**
1. Перейдите в "Настройки" → "Безопасность"
2. Включите "Автоматический выход"
3. Установите время бездействия
4. Сохраните настройки

### Уведомления о входах

**Настройка:**
1. Перейдите в "Настройки" → "Уведомления"
2. Включите "Уведомления о входах"
3. Выберите каналы доставки
4. Сохраните настройки

### Резервные коды

**Генерация:**
1. Перейдите в "Настройки" → "Безопасность"
2. Нажмите "Резервные коды"
3. Сгенерируйте новые коды
4. Сохраните коды в безопасном месте

## Часто задаваемые вопросы

### Можно ли войти с нескольких устройств?

Да, вы можете войти в аккаунт с нескольких устройств одновременно. Для безопасности рекомендуется регулярно проверять список активных сессий.

### Что делать, если забыл пароль?

Используйте функцию восстановления пароля на странице входа. Если это не помогает, свяжитесь с поддержкой.

### Как изменить email для входа?

1. Перейдите в "Настройки" → "Профиль"
2. Нажмите "Изменить email"
3. Введите новый email
4. Подтвердите изменение через почту

### Нужно ли каждый раз вводить код 2FA?

При использовании "Запомнить меня" код 2FA не требуется в течение 30 дней на том же устройстве. Для новых устройств код требуется всегда.
